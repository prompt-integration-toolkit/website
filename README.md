# Prompt-Integration-Toolkit Website

The central hub and official portal for the **Prompt-Integration-Toolkit** ecosystem.

This repository contains the source code for the web application, which serves as the project's main home. Here you will find:
- The official documentation and usage guides for the CLI tool.
- (Coming Soon) Public explorer for registered prompts.
- (Coming Soon) Showcase of users and platform contributors.

## 🛠 Technologies

- **Frontend:** React + Vite
- **Language:** TypeScript
- **Styling:** Vanilla CSS (Minimalist B&W Design)
- **Backend/DB:** Supabase
- **Hosting:** GitHub Pages

## 🚀 Local Development

To run the project on your machine:

1. Clone this repository:
   ```bash
   git clone https://github.com/prompt-integration-toolkit/website.git
   ```
2. Enter the directory:
   ```bash
   cd website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

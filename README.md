# Prompt-Integration-Toolkit Website

O hub central e portal oficial para o ecossistema **Prompt-Integration-Toolkit**. 

Este repositório contém o código-fonte da aplicação web, que serve como a casa principal do projeto. Aqui você encontrará:
- A documentação oficial e guias de uso da ferramenta CLI.
- (Em breve) Explorador público de prompts registrados.
- (Em breve) Vitrine de usuários e contribuidores da plataforma.

## 🛠 Tecnologias

- **Frontend:** React + Vite
- **Linguagem:** TypeScript
- **Estilização:** Vanilla CSS (Design Minimalista P&B)
- **Backend/DB:** Supabase
- **Hospedagem:** GitHub Pages

## 🚀 Desenvolvimento Local

Para rodar o projeto em sua máquina:

1. Clone este repositório:
   ```bash
   git clone https://github.com/prompt-integration-toolkit/website.git
   ```
2. Entre no diretório:
   ```bash
   cd website
   ```
3. Instale as dependências:
   ```bash
   npm install
   ```
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
